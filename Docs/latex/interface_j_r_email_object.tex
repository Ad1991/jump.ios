\begin{DoxyParams}{Parameters}
\item[{\em \_\-subject}]The desired subject of the email. The user can edit this value.\item[{\em \_\-messageBody}]The desired message body of the email. The message body can be in plain text or html, and if it is in html, this should be indicated by the argument isHtml. If you want to include urls that are shortened by to an \href{http://rpx.me}{\tt http://rpx.me} url, they should be added to the urls array, and the message body should contain the escape sequence @ where you want the urls to be substituted in. This value can be edited by the user.\item[{\em \_\-isHtml}]YES if the message body contains HTML content or NO if it contains plain text.\item[{\em \_\-urls}]The array of urls that \hyperlink{interface_j_r_engage}{JREngage} will shorten to an \href{http://rpx.me}{\tt http://rpx.me} url, in the order you wish the urls to be substitued into the message body, replacing the sequence @ in the order that the @'s appear.\end{DoxyParams}
