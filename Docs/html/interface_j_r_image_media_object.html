Detailed Description

Image object to be included in a post to a user's stream.

Create an image media object, fill in the object's fields, and add the object to the JRActivityObject::media array in your JRActivityObject. How the images get presented and whether or not they are used, depend on the provider.

Each image must contain a src URL, which maps to the photo's URL, and an href URL, which maps to the URL where a user should be taken if he or she clicks the photo.

See also:
Format and rules are identical to those described on the Facebook Developer page on Attachments.
