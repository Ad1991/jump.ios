Detailed Description

Mp3 object to be included in a post to a user's stream.

Create an mp3 media object, fill in the object's fields, and add the object to the JRActivityObject::media array in your JRActivityObject. How the mp3s get presented and whether or not they are used, depend on the provider.

Each mp3 must contain a src url, which is the URL of the MP3 file to be rendered. The mp3 can also include a title, artist, and album.

Note:
You can only include one JRMp3MediaObject in the media array. Any others will be ignored.
See also:
Format and rules are identical to those described on the Facebook Developer page on Attachments.
